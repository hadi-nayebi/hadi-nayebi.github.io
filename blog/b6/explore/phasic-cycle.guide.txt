Now, the big idea. When the seed agent does serious work, it does not just charge ahead. It moves through a fixed cycle of cognition compartments, and each one holds only one kind of thinking.

The cycle begins at idle, the resting state between jobs. From there it enters OBSERVE, where it gathers information by reading and searching, but writes no code yet. Next comes PLAN, where it turns what it gathered into an approach. Planning always sits between gathering and building. You cannot skip it.

Then EXECUTE, the only phase allowed to write project files, where the plan finally gets built. After that, VERIFY checks the work against the plan, with fresh eyes, unable to quietly fix what it finds. And finally CONDENSE, where the agent distills what it learned into lasting memory before returning to idle. This is where the brain actually grows.

Here is the heart of it. Each phase is only allowed to make its own kind of move. The tools for the other modes are simply withheld. OBSERVE cannot write code. EXECUTE cannot keep re-planning. Forbidding the other moves is the discipline. It holds the agent in the right state at the right time, so the work never blurs together.
